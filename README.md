# 🌤 Async Weather Tracker

A sleek, modern weather dashboard built with **Vanilla JavaScript**, featuring real-time data fetching, persistent search history, and a responsive **Glassmorphism** UI.

## ✨ Features

  * **Real-time Weather Data:** Fetches live data including temperature, humidity, and wind speed via the OpenWeatherMap API.
  * **Asynchronous Logic:** Uses `async/await` and the `fetch` API for non-blocking UI updates.
  * **Persistent History:** Saves your last 5 searches to `localStorage` (without JSON) so your data stays even after a page refresh.
  * **Glassmorphism UI:** A modern, frosted-glass aesthetic with CSS backdrop filters and fluid gradients.
  * **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.

## 🛠 Tech Stack

  * **HTML5** - Semantic structure.
  * **CSS3** - Custom properties, Flexbox, and Backdrop Filters.
  * **JavaScript (ES6+)** - Asynchronous functions and LocalStorage management.
  * **API** - [OpenWeatherMap API](https://openweathermap.org/api).
